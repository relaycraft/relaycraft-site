核心能力

流量监控

实时捕获、分析、筛选、导出 HTTP / HTTPS / WebSocket 流量

POST api.example.com/v2/users 200 OK
GET cdn.example.com/assets/logo.png 200 OK
GET api.example.com/v2/products?page=3 200 OK
POST api.example.com/v1/auth/login 401 Unauthorized
WS ws.example.com/live/stream 101
  • 多协议支持 — HTTP、HTTPS、WebSocket 流量一览无遗
  • 深度分析 — JSON 语法高亮、图片预览、时序 Waterfall
  • 智能筛选 — 按方法、域名、状态码、内容类型过滤
  • 一键导出 — cURL、HAR、Relay Session (.relay)
  • 断点调试 — 实时暂停、修改、恢复请求 / 响应
规则引擎

6 种规则类型

可视化规则构建器，无需配置文件，纯 UI 操作

📁

Map Local

将响应映射到本地文件，支持 JSON、图片、任意内容

🌐

Map Remote

将请求转发到不同 URL，一键切换测试环境 / 生产环境

🏷️

Rewrite Header

动态修改请求或响应的 Header，添加 Token、修改 UA

✏️

Rewrite Body

修改 Body 内容，支持 JSON Path、正则、纯文本替换

🐌

Throttling

模拟弱网环境，3G、Edge、自定义限速，测试用户体验

🚫

Block Request

即时拦截匹配请求，屏蔽广告、禁用特定 API 调用
